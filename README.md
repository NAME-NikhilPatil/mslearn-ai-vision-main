# Azure AI Vision Labs

This repository contains a series of labs designed to teach you how to use the Azure AI Vision service. Each lab provides hands-on experience with a specific feature of the service, with instructions for both C# and Python.

## Labs

Here is a list of the labs available in this repository:

*   **[Lab 1: Analyze Images](Instructions/Exercises/01-analyze-images.md)**: Learn how to analyze images to get insights, such as captions, tags, and objects.
*   **[Lab 2: Image Classification](Instructions/Exercises/02-image-classification.md)**: Train a model to classify images.
*   **[Lab 3: Custom Vision Object Detection](Instructions/Exercises/03-custom-vision-object-detection.md)**: Train a custom model to detect specific objects in images.
*   **[Lab 4: Face Service](Instructions/Exercises/04-face-service.md)**: Use the Face service to detect and analyze human faces in images.
*   **[Lab 5: Optical Character Recognition (OCR)](Instructions/Exercises/05-ocr.md)**: Extract printed and handwritten text from images.
*   **[Lab 6: Video Indexer](Instructions/Exercises/06-video-indexer.md)**: Analyze videos to extract insights.
*   **[Lab 7: Custom Vision Image Classification](Instructions/Exercises/07-custom-vision-image-classification.md)**: Train a custom image classification model.

## Prerequisites

Before you begin, you will need the following:

*   An [Azure subscription](https://azure.microsoft.com/free/).
*   [Visual Studio Code](https://code.visualstudio.com/).
*   The [Git command-line interface](https://git-scm.com/downloads).

## Getting Started

To get started, clone this repository to your local machine by running the following command:

```bash
git clone https://github.com/MicrosoftLearning/mslearn-ai-vision
```

Once the repository is cloned, open the folder in Visual Studio Code.

## Running the Labs

Each lab has its own set of instructions, which you can find in the `Instructions/Exercises` directory. The instructions will guide you through the process of setting up the necessary Azure resources and running the code.

For each lab, you can choose to use either C# or Python. The code for each language is located in the `Labfiles` directory, under the corresponding lab number.

Enjoy the labs!
